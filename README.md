# Cryptocurrency Creation

A Blockchain is a continuously growing list of records, called blocks, which are linked and secured using cryptography.

## Features
* Hash Cryptography
* Immutable Ledger
* Distributed P2P network
* Mining
* Consensus Protocol

## Examples of Cryptocurrency
* Bitcoin (BTC)
* Ethereum (ETH)
* Tether (USDT)
* Binance Coin (BNB)
* Solana (SOL)
* Cardano (ADA)
